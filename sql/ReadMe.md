# SQL Practice

Simple SQL practice environment for GitHub Codespaces.

## Quick Start

1. Click "Code" → "Codespaces" → "Create codespace on main"
2. Wait for setup to complete (10 seconds)
3. Run queries:

```bash
# Run all queries
./run

# Run specific query
./run 1

# List all queries
./run list

# Open SQLite shell
./run shell
